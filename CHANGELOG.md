# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-02-07

### Added
- Initial public release
- Clean Architecture backend with .NET 10
- CQRS pattern with MediatR
- Entity Framework Core with PostgreSQL
- Redis caching with hybrid strategy
- RabbitMQ event-driven messaging
- JWT authentication with refresh tokens
- Next.js 16 frontend with App Router
- AI-powered content assistance
  - Title generation
  - Content improvement
  - SEO description/excerpt generation
  - Tag recommendations
- Python FastAPI AI service with RAG
- ChromaDB vector storage
- Ollama integration for local LLM
- SignalR for real-time cache invalidation
- OpenTelemetry observability
- Rate limiting and security features
- Docker and docker-compose configuration

### Security
- JWT authentication
- Rate limiting per endpoint
- CSRF protection
- Input sanitization (HTML)
- Security headers
- Password complexity requirements
